iTunes library is a great media manager including books, music, movies, app and so on. And it plays a role in cleaning ebooks duplicates, fixing the eBook tag, adding artwork to eBooks. Okay! If epub books are unprotected from drm, just skip the step above and then learn how to side load unprotected files to the iTunes library and fix epub files. Make sure you have install iTunes library and run it. 

_**Tips:** the editions of my iTunes library is iTunes 11.3.0.54 and the computer operation system is win-7/64-bit operation system. As far as I know, If you want to add epub to iPad in **Mac OSX 10.9 Maverricks**, there is a bit problem. [So iTunes alternative can help us transfer epub books to iPad under Maverricks](https://tools.techidaily.com/epubor/products/)._ 

After launched iTunes library, go forward to "File" in the main menu and choose "Add File to Library" option. Then select the epub books where you have downloaded or saved, and click the _Open_ button.

![add epub file to iTunes](http://www.epubor.com/images/uppic/add-epub-books-to-iTunes-library.jpg)

Switch to _Books_ from the list of libraries, and if you find the imported eBooks are lack of covers, publisher name, etc; right-click the book title and select Get Info. The eBook metadata editor window will pop up.

![edit epub metadata via iTunes library](http://www.epubor.com/images/uppic/edit-ebook-metadata.jpg)

And we google the book title in the image.google.com and find the eBook cover. Once download this eBook cover to the local computer, switch to _Artwork_ option and browse the downloaded eBook cover. Finally click "Add" button. Then it automatically displays the eBook cover in the iTunes library.

![add cover to epub](http://www.epubor.com/images/uppic/add-cover-to-epub-via-itunes.jpg)

If you want to delete epub duplicates in iTunes library, hold on _the shift_ key and navigate to "View"->"Show Exact Duplicates Items". 

![show epub duplicates in iTunes](http://www.epubor.com/images/uppic/show-epub-duplicates.jpg)

### Load modified epub books to iPad iBooks

Plug your iPad into the computer and select the _Books_ tab. And tick "Sync Books" checkbox if it isn't already enabled. An "All books" option stands for transferring all epub books into the iPad. By checking a "Selected books" option, you can side load epub books to iPad as you want. 

Now select epub books as you want to load and press "Apply" button at the right bottom of the window.

![sync epub books to iPad ibooks](http://www.epubor.com/images/uppic/sync-books-to-ibooks.jpg)
