###### **Way 1\. Open Task Manager via taskbar**

``Right-click on the blank area on**taskbar**.``Then click**Task Manager**.```` ![](https://images.drivereasy.com/wp-content/uploads/2017/02/1-13.png) ````Now Task Manager is open.````

###### **Way 2\. Open Task Manager via Start Menu**

````Click**Start**button, scroll down to locate and click**Windows System**.``Then click**Task Manager**.`` `` ![](https://images.drivereasy.com/wp-content/uploads/2017/02/4-7.png) `` `` Now Task Manager is open. `` ``

###### Way 3\. Open Task Manager via Quick Access Menu

``Open**Quick Access Menu**by right-clicking on**Start**button.``Then click**Task Manager**.```` ![](https://images.drivereasy.com/wp-content/uploads/2017/02/3-7.png) ````Now Task Manager is open.````

###### **Way 4\. Open Task Manager via Run dialog box**

``Open **Run**  dialog box by pressing **Win + R** (Windows logo key and R key) at the same time.``Then Type **taskmgr** into the box and hit **Enter**.```` ![](https://images.drivereasy.com/wp-content/uploads/2017/02/5-5.png) ````Now Task Manager is open.````

###### **Way 5\. Open Task Manager via Ctrl+Alt+Del option**

``Press **Ctrl+Alt+Del**.``Then click**Task Manager**on the pop-up window.```` ![](https://images.drivereasy.com/wp-content/uploads/2017/02/2-13.png) ````Now Task Manager is open. Try it by yourself now. And if you found this really useful, please share it with you friends, many thanks!```` ````
