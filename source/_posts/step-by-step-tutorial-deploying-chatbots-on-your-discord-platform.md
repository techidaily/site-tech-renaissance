## Step-by-Step Tutorial: Deploying Chatbots on Your Discord Platform

Close 

 The number of[ Discord](https://www.lifewire.com/discord-4174046) bots available for download continues to grow due to an active developer community. If you know how to add bots to Discord, there's a wide array of functionality you can take advantage of, from bots that play music to ones that facilitate donations.

##  How to Add Bots to Discord 

 Before you can add bots, you must[ create a Discord server](https://www.lifewire.com/make-a-discord-server-5074624) . If you plan to add a bot to a server that's not yours, confirm that you've been granted permission to manage the server. These permissions are automatically in place for any server that you create.

1. [ Visit Discord Bots at top.gg](https://top.gg/) in your web browser.
2. Search for bots by keyword or browse the options available. Select**View** to learn more about a bot and add it to a server.  
![A list of Discord bots at top.gg](https://www.lifewire.com/thmb/03We2UoLRgRpj6FzKVziCOMvAvs=/1500x0/filters:no_upscale():max_bytes(150000):strip_icc():format(webp)/discord-bots-list-ea4e75a4c75b4a2ba8025134a5234b39.png)
3. Select**Invite** . Discord launches automatically.  
![A description for the Green-bot Discord bot](https://www.lifewire.com/thmb/V9r87j8WHIz9WCZdYZPmJFNumNU=/1500x0/filters:no_upscale():max_bytes(150000):strip_icc():format(webp)/green-bot-discord-20c5e5c497f441a3a44fccd532a880ca.png)
4. Select a server from the drop-down menu, then choose**Continue** .  
[ How to Find Discord Servers ](https://www.lifewire.com/how-to-find-discord-servers-5092039)  
![A Discord server chosen for a Discord bot](https://www.lifewire.com/thmb/3u7qRVUiYYFSMOFCznp2fV_Jw_w=/1500x0/filters:no_upscale():max_bytes(150000):strip_icc():format(webp)/discord-bot-choose-server-5bd1c64640084f969e3944eba2a12ac8.png)  
 If you're not logged in to Discord, you'll be prompted to do so now.
5. Ensure that all the permissions are selected, then select**Authorize** .  
![The Authorize button for a Discord bot](https://www.lifewire.com/thmb/7rycQakx-qIQHL_CKQTCfIzKsRc=/1500x0/filters:no_upscale():max_bytes(150000):strip_icc():format(webp)/authorize-discord-bot-5348d152d14c446e9db299f70f8dd752.png)  
 You can deny a certain set of permissions by removing its check mark, but that will likely cause the bot to malfunction in certain instances.
6. Select the**I am human** check box.  
![The human confirmation check on Discord](https://www.lifewire.com/thmb/y_CcQKYfyCUllVffdgl-IKCDYF4=/1500x0/filters:no_upscale():max_bytes(150000):strip_icc():format(webp)/discord-bot-human-check-bc7b7016b6924e089477b885b4cbb581.png)
7. A message appears confirming the bot is authorized and added to your server. Select**Go to \[server name\]** or close the tab or window to complete the process.  
![The Green-bot Discord bot](https://www.lifewire.com/thmb/d4o4wYddmYKoUgHC5_wRKIqo5FM=/1500x0/filters:no_upscale():max_bytes(150000):strip_icc():format(webp)/green-bot-discord-bot-5056853ec8ce452c891f0ddc2a3d4a6e.png)

##  Other Discord Bot Websites 

 Top.gg isn't the only place to find bots for your server. Other bot websites include:

* [ Bots on Discord](https://bots.ondiscord.xyz/) : Useful tags and a Most Used Bots section make it easy to peruse some of the best options.
* [ Carbonitex](https://www.carbonitex.net/discord/bots) : This site's interface is a bit different from its competitors. Still, it provides access to a large number of feature-rich server additions.
* [ Discord Bots](https://discord.bots.gg/) : This easy-to-navigate repository of over 2,000 bots is worth browsing if you're in the market for some new additions to your server. A filter makes it a cinch to locate only open-source bots.

 As with any unregulated software, use caution when selecting a bot provider. High-trafficked sites with a significant number of reviews are typically more reputable than some of their lesser-known counterparts.

##  Adding Your Own Discord Bots to a Server 

 It's also possible to[ create your own Discord bots](https://www.lifewire.com/make-a-discord-bot-4767916) and add them to a server instead of those developed by a third party. This gives you the freedom to customize them using JavaScript code and server-designated permissions. For example, you can set up an automated moderator bot that automatically bans problematic users.
